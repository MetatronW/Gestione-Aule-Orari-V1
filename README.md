
# Gestionale Orari Aule Scolastiche e Universitarie

Applicazione Web sviluppata in Flask per la gestione di aule e laboratori scolastici o universitari che permette a docenti e amministratori di:
   - Prenotare aule/laboratori per date e fasce orarie specifiche
   - Visualizzare la griglia oraria delle prenotazioni
   - Pubblicare e visualizzare avvisi/comunicazioni
   - Gestire accessi con ruoli diversi (pubblico, docente, admin)

### Sicurezza

La parte di sicurezza delle password è stata tralasciata nello sviluppo del progetto per permettere di concentrarsi sulle funzionalità di prenotazione/visualizzazione degli orari delle lezioni.

La funzionalità di hashing delle password sarà implementata in futuro con strumenti come Bcrypt o Werkzeug security module.
